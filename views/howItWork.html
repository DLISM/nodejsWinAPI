Реализация

Пользователей в Windows получаю с помощью C++. Реализация находится в файле get_users.cc и подключается к Node.js как модуль.

В Node.js вызывается метод getUsers(), который возвращает всех пользоватлей в виде строки.

Получанная строка разбывается на массив и сравнивается с входящим параметром

Для корректной сборки нужны средства для разработки С++ и Pyton3
